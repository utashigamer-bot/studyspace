📲 Installer StudySpace

Ajoute l'app sur ton écran d'accueil pour l'ouvrir comme une vraie appli, sans navigateur.

1
Ouvre ce fichier dans Safari
Assure-toi d'utiliser Safari (pas Chrome ni Firefox). Tu peux envoyer le fichier .html par AirDrop ou e-mail.
🧭
2
Appuie sur le bouton Partager
C'est l'icône carrée avec une flèche vers le haut, en bas de l'écran (iPhone) ou en haut (iPad).
⬆️
3
Tape "Sur l'écran d'accueil"
Fais défiler le menu et appuie sur "Ajouter à l'écran d'accueil". Donne-lui le nom "StudySpace".
➕
4
C'est prêt ! 🎉
L'icône apparaît sur ton écran d'accueil. Toutes tes données sont sauvegardées dans Safari automatiquement.
✅

Ouvre le fichier dans Safari
Double-clique sur le fichier espace-etude-v3.html — il s'ouvrira automatiquement dans Safari.
🗂️
2
Menu Fichier → Ajouter au Dock
Dans Safari, clique sur "Fichier" dans la barre de menus, puis "Ajouter au Dock…" (macOS Sonoma+).
🖥️
3
Donne-lui un nom et confirme
Nomme-le "StudySpace" et clique sur Ajouter. L'icône apparaîtra dans ton Dock comme une vraie app.
✏️
4
C'est prêt ! 🎉
Clique sur l'icône dans le Dock pour ouvrir StudySpace en plein écran, sans barre de navigation.
